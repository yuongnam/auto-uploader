## 概览

本系统由两部分组成：

1. **云函数**（定时运行）
   - 抓取 YouTube/TikTok 热门搞笑视频
   - 下载、去重（FFmpeg）
   - 上传到阿里云 OSS
   - 更新 `publish_queue.json` 队列

2. **手机端 Auto.js 脚本**
   - 轮询 `publish_queue.json`
   - 下载视频
   - 打开抖音/快手 App，模拟发布

---

## 快速开始

1. 准备阿里云 OSS（Bucket public-read）
2. `cd auto-uploader/cloud && vercel --prod`
3. 修改 `config.json` 填写你的 OSS 信息
4. 手机安装 Auto.js Pro，导入 `android/auto_publisher.js`，修改 `QUEUE_URL`
5. 运行测试

---

## 坐标校准（重要）

脚本中使用相对坐标（0-1）。如果你的手机分辨率不同，需要调整：

- 底部「+」按钮: `click(0.5, 0.95)`
- 视频选择: `click(0.3, 0.3)`
- 下一步: `click(0.85, 0.95)`

建议用 `ui` 控件查找代替绝对坐标，详见 [02-tips.md](02-tips.md)。
